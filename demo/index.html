Demo of Viewport Class jquery.viewport_class.js

jQuery plugin (for responsive design) registers an element to maintain a css class of the viewport when it changes (with optional callback on viewport change)

Download jquery.viewport_class.js

Instructions

  1. Reize your browser window to see the viewport change callback fire.
  2. Inspect the h1 element and notice the class changes to match the viewport.
